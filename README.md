# KES Updates — Multi-Page Student Portal

A multi-page student portal for KES (KES' Shroff College) with notices, academic resources, ERP links, library access, attendance calculator, and more.

## 📁 File Structure

```
kes-updates/
├── index.html          🏠 Home / Dashboard
├── notices.html        📢 Degree & Junior College Notices
├── academic.html       📚 Academic Calendar, Fees, Cutoff
├── papers.html         📄 Previous Year Question Papers
├── erp.html            💳 ERP Portal Links
├── library.html        📚 Library Catalogue
├── attendance.html     🧮 Smart Attendance Calculator
├── college.html        🏛️ College Details & Contacts
├── student.html        🧑‍🎓 Student Corner (Memes, Doubts, Issues)
├── collab.html         🤝 Collaboration
├── about.html          ℹ️ About KES Updates
├── policy.html         📄 Public Disclaimer & Policy
├── styles.css          🎨 Shared Styles (Light/Dark Mode)
├── script.js           ⚙️ Shared JavaScript
└── attendance.js       🧮 Attendance Calculator Logic
```

## 🚀 How to Upload to GitHub Pages

1. **Create a new repository** on GitHub (e.g., `kes-Updatess`)
2. **Upload all files** from this folder to the repository root
3. Go to **Settings → Pages** in your repo
4. Select **Deploy from a branch** → **main** → **/(root)**
5. Your site will be live at: `https://kesupdatess.github.io/kes-Updatess/`
